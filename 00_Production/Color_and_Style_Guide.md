# TEN LIFETIMES — Color & Style Guide (working reference)

> Per-chapter grade so each chapter is identifiable from color alone. **Canonical in `01_Production_Bible.md` §7.**

## Color script
| Chapter | Dominant color | Grade / atmosphere |
|---|---|---|
| Cosmos / Intro | Deep blue/black + silver | Starlight, void, brass-gold glints on clockwork |
| Egypt | Gold | Warm, eternal, low-sun, sand haze |
| Rome | White marble | Noble, bright, clean highlights |
| Viking | Blue-gray | Cold, desaturated, windswept, aurora accents |
| Medieval | Amber | Firelight and stone, warm pools in shadow |
| Renaissance | Rich burgundy | Romantic, candlelit, painterly |
| Frontier | Orange | Dust, sunsets, freedom (Arizona-style) |
| Victorian | Emerald | Sophisticated, nostalgic, gaslight |
| WWII | Muted olive | Desaturated, hope amid uncertainty |
| Modern | Natural | Authentic, ungraded-feel, grounded |
| Future | Silver & blue | Peaceful, clean, timeless, soft glow |
| **Real photo ending** | **Natural / ungraded** | The deliberate visual exhale — reality takes over |

## Transition-color rule
Transitions **morph the palette** from one chapter into the next (gold→marble-white, snow→ash→petals→prairie-dust→train-steam). See Bible §10 "history transforms."

## The Bridge
Abandons all chapter palettes → deep cosmic blue/black + starlight silver. Reference tone: *Interstellar, The Fountain, Cloud Atlas, What Dreams May Come* (Risk R-10 watch).

## Motifs (`18_Motifs\`)
Silver pendant (primary), wedding ring, pocket watch, stars, clock gears, dust particles, constellations, roses, ancient-map textures — generated once as clean reusable plates, reintroduced across chapters and the Bridge.
