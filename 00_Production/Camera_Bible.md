# TEN LIFETIMES — Camera Bible (working reference)

> Lens & movement language. **Canonical principle in `01_Production_Bible.md` §9 (the Emotional Wave).** This expands it into a usable grammar.

## Core rule
Camera scale follows the song's emotional intensity. **Never** move dramatically during intimate lyrics; grow visual scale as the orchestra grows.

## Lens grammar
| Lens | Use | Feeling |
|---|---|---|
| 85mm | Emotional close-ups, the morph, the Bridge reunion | Intimacy, compression, "just them" |
| 50mm | Intimate two-shots, gestures | Natural, present |
| 35mm | Environmental wides, establishing | Place, scale, era |
| wide/anamorphic | Final-chorus convergence, cosmos | Epic, sweeping |

## Movement vocabulary
- **Slow push / dolly-in** — verses, intimacy, the morph to real photo.
- **Slow orbit** — gestures (let the silver pendant catch light).
- **Crane / rise** — chorus lifts, as strings swell.
- **Float / drift (gravity off)** — the Bridge (cosmic, dreamlike).
- **Static hold** — the final real photograph (slow zoom only).
- **Match-on-gesture / morph** — chapter-to-chapter transitions (Bible §10).

## Emotional Wave mapping
```
Intro    : fade-in, slow dolly (stars, gears)            — scale 1
Verse 1/2: 50–85mm, slow push, minimal move              — scale 2–4
Pre-chorus: begin widening as strings rise               — scale 5
Chorus 1/2: crane/rise, wide swell, impacts              — scale 6–8
Bridge   : drums out → intimate → cosmic float           — dip then surge
Final ch.: maximum scale, sweeping, convergence          — scale 10
Outro    : back to intimate push → morph → static hold   — scale 3 → land
```

## Standing rules
- 16:9 (2.39:1 letterbox optional). Default to motivated, single moves per shot (Sora handles one clean move best).
- Clip length ≤ tool max (Sora ≈ 20s). Design shots to that limit; prefer still→subtle-motion for hero faces (lowers artifact risk — Risk R-02).
